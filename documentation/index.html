D) Email Settings

An user can contact you using contact form in the footer. To set e-mail address where you want to receive e-mails and other settings open contact.php located in root directory.

contact.php settings you can change: 
$to = 'hithemestarz@gmail.com'; // Here you can set address for receiving emails (usually your email)
$subject = 'Message From Your Website Contact Form'; // Message displayed in "subject" label in email

Email address for reply

Direct reply to users email
You can also change, how you want to reply to email. You can set the email "from" to be the email which user has provided in contact form, so you can directly reply him, just to click on "reply" button in your email interface. The code is this:

$headers = 'From: ' .$email . "\r\n"; // Email "from" the same as user has provided. Default state

One incoming email address or "noreply" address
Or you can use one address widely known as "noreply@yourdomain.com" and users email address will be displayed in body of the message. Then you need to uncomment (remove "//" dashes) of the commented code and comment the code above. The final code should look like this:

//$headers = 'From: ' .$email . "\r\n";

$headers = 'From: noreply@domain.com' . "\r\n";

$body .= "";
$body .= "Email: ";
$body .= $email;
$body .= "\n";

Note: Your hosting provider has to support PHP in order to make the contact form work.

E) CSS Classes You Can Use in HTML

ANAGLYPH comes with several prepared CSS classes you can use right in HTML without coding it into CSS. This is helpful if you want to style only one element without affecting others. For example, you have tree headings, but you want only first to be in the white color, second one with opacity and third one will have subheading, so you don't want to have big gap under. HTML code will look like:

< h2 class="font-color-light">Heading One</h2>
< h2 class="has-opacity">Heading Two</h2>
< h2 class="has-subtitle">Heading Three</h2>

You can use these predefined classes:

  • center - center everything in element
  • font-color-light - font will be in white color
  • font-color-dark - font will be in primary (dark) color
  • background-color-grey-dark - set background color to dark (from _variables.scss)
  • background-color-grey-light - set background color to white
  • background-color-grey-footer - set background color to light grey (from _variables.scss)
  • background-color-primary - set background color to primarycolor (from _variables.scss)
  • background-color-secondary - set background color to secondary color (from _variables.scss)
  • center - center elements
  • font-color-light - set font color to white
  • font-color-dark - set font color to dark (from _variables.scss)
  • has-opacity - set opacity to element

  • has-parallax - required if you want to use parallax effect for background images
  • menu-child-align-left - align rolled menu to left
  • menu-child-align-right - align rolled menu to right (use for last menu item)

F) How to Create Anaglyph Images

How to add Theme Starz - Anaglyph action into Photoshop
Just double click on "Theme Starz - Anaglyph.atn" and Photoshop should automatically add this new action to the "Actions" pannel. If not, copy and paste this "Theme Starz - Anaglyph.atn" to *PHOTOSHOP-PATH*/Presets/Actions and start Photoshop.

How to use Theme Starz - Anaglyph action

  1. Open your image
  2. Navigate to Window > Actions
  3. Click on Theme Starz - Anaglyph
  4. Wait for the result
  5. Save your new image
